# LocalPro — Claude Code Master Guide

**LocalPro** — mobile marketplace for local services. Providers list on a map; clients search by location/category/radius, then chat.

| Layer | Stack |
|---|---|
| Backend | Java 21 · Spring Boot 3.3.6 · `com.localpro` · Lombok 1.18.40 (pinned) |
| Frontend | Flutter 3 · Riverpod 2 |
| Services | Firebase Auth/FCM · Supabase PostgreSQL + PostGIS · Cloudinary · Railway |
